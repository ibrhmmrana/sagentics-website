# Images folder

Put all site images here. You can use subfolders (e.g. `hero/`, `team/`, `logos/`).

**In code:**
- Use the path from the site root: `/images/your-file.jpg` or `/images/hero/banner.jpg`
- With Next.js `<Image>`: `src="/images/your-file.jpg"` and set `width` and `height` (or use `fill` for responsive)
